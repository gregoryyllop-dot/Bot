🛠️ SEIMI - ESPACE MODÉRATION

Documentation officielle et outils réservés au personnel du serveur

📄 Commandes de Modération

!config

Ouvre le panneau de configuration interactif (Boutons et sélecteur de rôles) pour modifier les paramètres système du bot et accéder au menu de Modération.

!clear [1-100]

Supprime instantanément le nombre de messages spécifié dans le salon actuel (ignore automatiquement les messages datant de plus de 14 jours).

!mute @membre [temps]

Exclut temporairement un utilisateur du serveur. Les durées acceptées par le système sont : 1m, 5m, 10m, 30m, ou 1h.

!setupcodes

Installe le message d'accueil fixe avec le bouton cliquable secret permettant aux membres de récupérer leurs codes Roblox de manière éphémère.

!ban @membre

Bannit définitivement un membre du serveur. Le bot demande une confirmation explicite (oui/non) dans le salon pour éviter les miss-clics du staff.

⚖️ Charte d'utilisation du Staff

  • Exemplarité : Les modérateurs doivent respecter le règlement du serveur à la lettre.
  • Sécurité : Ne partagez jamais l'accès aux commandes d'administration ou aux configurations sensibles.
  • Propreté : Veillez à utiliser les fonctionnalités de suppression automatique pour garder les salons lisibles.
